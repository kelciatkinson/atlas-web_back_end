# 🖥️ Atlas Web Back End

A full-spectrum backend development portfolio built during 
Atlas School — covering Python async, authentication systems, 
caching, databases, and API design.

## 🚀 Key Highlights
- ⚡ Async Python pipelines with asyncio and concurrent coroutines
- 🔐 Multi-layer auth: Basic, Session, Token, and Flask user services
- 🧠 Caching systems implementing FIFO, LRU, LFU, and MRU policies
- 🛡️ PII protection with log filtering and password encryption
- 🗄️ Advanced MySQL (procedures, triggers, indexes) + MongoDB NoSQL
- 🌍 Flask i18n internationalization and localization
- 🧪 Unit and integration testing with mocking and fixtures

## 🛠️ Tech Stack
Python · Flask · Node.js · Express · Redis · 
MongoDB · MySQL · asyncio · Babel · Nodemon

## 💡 AI & Automation Relevance
Async programming patterns and caching architecture in this 
project mirror the infrastructure behind modern AI pipelines, 
background job processing, and intelligent API systems.
